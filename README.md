# Grafana Cloud Labs

Repositório utilizado para armazenar exemplos, laboratórios e testes relacionados ao Grafana Cloud.

## Projetos

### fastapi-test

Exemplo de aplicação FastAPI instrumentada com OpenTelemetry e Grafana Alloy para envio de traces ao Grafana Cloud.

Funcionalidades:

* Endpoint `/health`
* Endpoint `/minha-api`
* Integração com OpenTelemetry
* Exportação de traces via Grafana Alloy
* Ambiente local utilizando Python Virtual Environment (venv)

### grafana-o11y

Exemplos e configurações relacionados a observabilidade utilizando Grafana Cloud.

## Objetivo

Centralizar projetos de estudo, testes de observabilidade, instrumentação de aplicações e integrações com Grafana Cloud.

## Tecnologias

* Grafana Cloud
* Grafana Alloy
* OpenTelemetry
* FastAPI
* Python
* Linux / WSL
* GitHub
